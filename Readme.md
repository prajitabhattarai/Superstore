Superstore Data Analysis Project

    /**********   Project Overview  ************/

This project focuses on analyzing the Superstore dataset using Python to understand sales and profit performance. The project involves data cleaning, exploration, and visualization to identify important business insights and trends.The main purpose of this project is to learn how data analysis can help businesses make better decisions and improve overall performance.

In this project, I performed: 
    - Data loading
    - Data cleaning
    - Exploratory Data Analysis (EDA)
    - Data visualization
    - Insight generation

    /*************  Objectives *************/

The objectives of this project are:
    - To explore and understand the dataset
    - To analyze sales and profit performance
    - To visualize data using graphs and charts
    - To gain practical experience in data analysis using Python

Dataset Information:
    The dataset used in this project is the “Superstore Dataset” which is in CSV format.

The dataset contains information such as:
    - Order details
    - Product categories
    - Sales
    - Profit
    - Discounts
    - Customer segments
    - Regions and states

Tools and Libraries Used
Programming Language
- Python 3

Libraries
- Pandas
- NumPy
- Matplotlib
- Seaborn

    /****************** Project Workflow **************/

1. Data Loading
    The dataset was loaded using the pandas library.

2. Data Exploration
    Basic information about the dataset was checked using:
    - 'shape'
    - 'columns'
    - 'info()'
    - 'describe()'

3. Data Cleaning
    - Checked missing values
    - Removed duplicate records

4. Exploratory Data Analysis (EDA)
    Analysis was performed on:
    - Sales by category
    - Profit by category
    - Sales by region
    - Sales over Time
    - Relationship between discount and profit
    - Segment-wise Sales

5. Data Visualization
    Different charts and graphs were created to better understand the data.
        Examples:
            - Bar charts
            - Heatmaps
            - Category comparison graphs


    /*********************** Key Insights ********************/

    - Technology category generated high profit compared to other categories.
    - Some categories had high sales but lower profit.
    -  The South region had better sales performance.
    - Discounts may reduce overall profit.
